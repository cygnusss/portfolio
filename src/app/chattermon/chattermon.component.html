Chattermon: Gotta Chat'em All

Chattermon is a terminal based Pokemon battle system. Players create an account, then start of join a game. Games consist of two players, each assigned three Pokemon at random. A player wins by defeating each of their opponents Pokemon. Players can also chat with each other.

dev stack¬

Reactjs + Routing, Node + Expressjs, PostgreSQl + Sequelize, PassportJS, bcrypt
